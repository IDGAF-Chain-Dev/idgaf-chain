## Adding to MetaMask

### Manual Addition

1. Open MetaMask
2. Go to Settings → Networks → Add Network
3. Enter the following:

```
Network Name: IDGAF Chain
RPC URL: https://rpc.idgaf.chain
Chain ID: 10144
Currency Symbol: IDGAF
Block Explorer URL: https://explorer.idgaf.chain
```

### Chainlist

Add to [Chainlist](https://chainlist.org) for easy discovery.
